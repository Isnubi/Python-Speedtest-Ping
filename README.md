<!-- GETTING STARTED -->
## Getting Started
<a name="getting-started"></a>

You can install the requirements by following these steps.

### Prerequisites

1. Clone the repository on your computer.

    ```sh
    git clone https://github.com/Isnubi/Python-Speedtest-Ping.git
    cd Python-Speedtest-Ping
    ```
   
   * If **Git** is not installed, you can install it using the following command or 
   download the repository as a zip file from [here](https://github.com/Isnubi/Python-Speedtest-Ping/archive/refs/heads/master.zip)
        ```sh
        sudo apt install git
        ```

2. Launch the install.sh file to install the required packages.

    ```sh
    sudo chown +x install.sh
    ./install.sh
    ```

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- USAGE EXAMPLES -->
## Usage

To use the script, you can use the following command:

```sh
sudo python3 main.py
```

It will print you the results in the terminal in JSON format.
